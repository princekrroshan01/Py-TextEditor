# Py-TextEditor
    A pure python based text editor

## using
	    
```python        
git clone https://github.com/princekrroshan01/Py-TextEditor/
cd Py-TextEditor
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt

then run run.py
```
